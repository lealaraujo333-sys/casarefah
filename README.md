# Casa Refah

E-commerce de velas artesanais, difusores aromáticos e objetos decorativos em cimento.

## Stack

**Frontend**: React + Vite + TypeScript + TailwindCSS  
**Backend**: Hono + Prisma + MongoDB  
**Deploy**: Vercel (frontend) + Render (backend)

## Desenvolvimento Local

```bash
# Frontend
npm install
npm run dev

# Backend  
cd backend
npm install
npm run dev
```

## Produção

- **Frontend**: https://casarefah.vercel.app
- **Backend**: https://casarefah.onrender.com

## Admin

Acesse `/admin/login` para gerenciar produtos e configurações.
